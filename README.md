## Cloning the repository

```shell
git clone https://github.com/MexLegend/angular-real-state-app.git
```

## Getting Started

Install the dependencies:

```sh
$ pnpm i
# or
$ yarn
# or
$ npm i
```

Setup .environment file

```js
URI: 
MAPBOX_TOKEN: 
CLOUDINARY_NAME:
CLOUDINARY_PRESET:
```

Start the app

```sh
$ pnpm start
# or
$ yarn start
# or
$ npm run start
```

## Built With

- Angular 16.1.0
