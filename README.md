## Cloning the repository

```shell
git clone https://github.com/MexLegend/netflix-clone.git
```

## Getting Started

Install the dependencies:

```sh
$ pnpm i
# or
$ yarn
# or
$ npm i
```

Setup .env file

```js
DATABASE_URL=
NEXTAUTH_SECRET=
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
GITHUB_ID=
GITHUB_SECRET=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

Start the app

```sh
$ pnpm start
# or
$ yarn start
# or
$ npm run start
```
